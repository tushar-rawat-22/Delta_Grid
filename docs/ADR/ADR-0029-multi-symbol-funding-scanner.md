## Status

## Context

Mission 28 proved that weak ETHUSDT funding can be rejected after costs.

The next profitable direction is not optimizing one weak symbol.

The system needs to scan many liquid perpetual markets and identify where funding edge is actually large enough after cost assumptions.

This mission still does not execute trades.

## Scanner Logic

The scanner evaluates:

- current funding rate
- annualized funding rate
- mark price
- index price
- basis percentage
- 24h price change
- 24h quote volume
- open interest
- open interest value
- expected funding edge
- cost proxy
- net expected edge
- edge-to-cost ratio
- liquidity score
- funding score
- basis risk penalty
- scanner score
- GO / NO-GO verdict
