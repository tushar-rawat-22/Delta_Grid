# DeltaGrid Charter

Mission:
Build a risk-first DeFi research and execution platform.

Rules:
1. No real capital in early phases.
2. No private keys in committed files.
3. No sandwiching or harmful MEV.
4. Every strategy must pass simulation before execution.
5. Every execution must pass onchain invariant checks.
6. Capital preservation is more important than trade frequency.

Promotion Gates:
Research -> Local simulation -> Testnet -> Fork testing -> Paper/demo -> Tiny live capital -> Scaled capital.
